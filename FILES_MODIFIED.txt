================================================================================
KEY SETTINGS & DATABASE CHANGES
================================================================================

Database Setting:
- Table: oc_setting
- Key: config_remove_url_prefixes
- Type: integer
- Default: 0 (disabled)
- Values: 0 = Off, 1 = On

Configuration Flow:
1. Admin toggles setting in UI (Settings > Settings > Server)
2. Setting saved to oc_setting table
3. catalog/controller/startup/seo_url.php reads setting
4. URL generation respects setting value
5. .htaccess handles 301 redirects
